# SafePic

Простая программа на Python для защиты изображений. Использует алгоритм шифрования XOR с хешированием пароля по стандарту SHA-256.

## Возможности
- Шифрование файлов: Превращает изображения в защищенный формат .locked.
- Автоматическое удаление: Исходный файл удаляется после успешного шифрования.
- Выделенная папка: Все результаты сохраняются в директорию SAVEPIC.
- Простой графический интерфейс на базе библиотеки tkinter.

## Инструкция по использованию
1. Запустите скрипт.
2. Нажмите кнопку ОБЗОР и выберите нужный файл.
3. Введите секретный пароль в соответствующее поле.
4. Нажмите кнопку ЗАШИФРОВАТЬ. Исходное изображение будет удалено, а защищенная копия появится в папке SAVEPIC.
5. Для восстановления выберите .locked файл, введите пароль и нажмите РАСШИФРОВАТЬ.

## Технические требования
Для запуска требуется Python версии 3.x. Программа использует только стандартные библиотеки: tkinter, os, hashlib. Дополнительная установка зависимостей не требуется.

## Предупреждение
В случае утери пароля восстановить данные невозможно. Будьте внимательны при вводе и хранении паролей.
